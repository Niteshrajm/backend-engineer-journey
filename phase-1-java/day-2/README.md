# 📅 Day 2 - Operators & Conditional Statements

---

## 📌 Topics Covered
- Arithmetic Operators (+, -, *, /, %)
- Relational Operators (==, !=, >, <, >=, <=)
- Logical Operators (&&, ||, !)
- Assignment Operators (+=, -=, etc.)
- User Input using Scanner
- Conditional Statements (if, if-else, if-else-if)
